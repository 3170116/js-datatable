columns: array css: object data: array pagers: array size: integer type: string
columns Required
array

Describes meta data for each column. Each element is an object with:
'header' : the title to display to each column
'sortable' : defines if user can sort data based to this column (default value true)
'type' : the type of the data to each column ('string' or 'number')

css Optional
object

Holds css classes for table elements.
Available properties: 'table'

data Optional
array

Stores the data only when type is 'client'. Each element is an array containing the data for each column.

pagers Optional
array

Stores the available sizes.
Value -1 means All.

size Required
integer

Specifies the number of rows which will be visible when the table is initialized.

type Required
string

Specifies the source of data.
The value can be 'client' or 'server'
